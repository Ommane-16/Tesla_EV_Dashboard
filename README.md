# ⚡ Tesla EV Dashboard (Power BI Project)

🚀 **Interactive Power BI dashboard analyzing Tesla’s EV ecosystem** — covering model performance, revenue trends, charging infrastructure, and environmental impact.

---

## 📊 Project Overview

This project presents a comprehensive analysis of Tesla electric vehicles using Power BI.
It focuses on understanding:

* 🚗 Model-wise performance
* 📈 Revenue & production trends
* ⚡ Charging infrastructure growth
* 🌱 Environmental impact (CO₂ savings)

The dashboard is designed with **interactive visuals, slicer buttons, and clean UI** for better insights.

---

## 🧩 Dashboard Pages

### 🔹 1. Infrastructure Overview

* Charging stations growth
* Net CO₂ saved
* Region-wise infrastructure distribution

---

### 🔹 2. Production & Revenue

* Total production analysis
* Revenue trends
* Revenue per charging station
* Year-wise financial insights

---

### 🔹 3. Model Insights

* Model-wise KPIs (Price, Revenue, Rank)
* Deliveries vs CO₂ impact
* Region-wise revenue analysis
* Interactive model selection (buttons)

---

### 🔹 4. Model Comparison

* Final summary table
* Model-wise comparison
* Key metrics (Revenue, Range, CO₂, Rank)

---

## 📁 Dataset Description

The dataset includes the following key attributes:

* Tesla Models: Model S, Model 3, Model X, Model Y, Cybertruck
* Revenue and Production data
* Deliveries and fulfillment metrics
* Charging stations data
* Battery capacity and range
* CO₂ emissions and savings
* Regional distribution

---
